
# Chatbot Mariage n8n

Projet de chatbot intelligent conçu pour automatiser la gestion des invités lors d’un mariage.

 Objectif du projet

Simplifier l’organisation d’un événement en automatisant :
- L’enregistrement des invités
- La centralisation des données
- La gestion des confirmations

 Fonctionnalités

- Enregistrement automatique des invités via webhook
- Réponses automatisées aux questions sur l’événement
- Stockage des données dans Google Sheets
- Workflow entièrement automatisé avec n8n

 Technologies utilisées

- n8n (Automation Workflow)
- Webhooks
- Google Sheets API
- Logique conditionnelle (IF / Switch)

 Architecture

1. Réception des données via Webhook  
2. Traitement des informations  
3. Enregistrement dans Google Sheets  
4. Réponse automatique à l’utilisateur  

 Import du projet

1. Télécharger le fichier JSON
2. Ouvrir n8n
3. Importer le workflow
4. Configurer les identifiants (Google Sheets / Webhook)

---

Projet réalisé dans une logique d’automatisation événementielle et de gestion intelligente des flux de données.
